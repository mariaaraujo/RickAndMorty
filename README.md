## Getting Started

Para iniciar a aplicação, com o node intalado utilize o comando:

```bash
npm install
```

Logo depois
```bash
npm install -g yarn
```

E rode a aplicação em dev com:

```bash
yarn dev
```

No navegador acesse: [http://localhost:3000](http://localhost:3000) para visualizar a aplicação rodando
